How to connect this page to the API

Serve this page from the same origin as the backend (for example open it via FastAPI static files or uvicorn on port 8000), or point the client at your server using data-api-url on the <html> element or ?api= (full URL), e.g. http://127.0.0.1:8000/predict.

The parser accepts a flat JSON body (risk, soil_moisture, …) or the FastAPI shape with data.moisture. If /predict returns {"error":"No sensor data available"}, the status line shows Warning (API reachable) and the subtitle shows the server message until valid data exists.
