About

This project focuses on predicting landslide risks by analyzing soil conditions and weather patterns using sensor data and machine learning. By combining environmental monitoring with AI-driven analysis, the system provides early risk detection to improve safety and preparedness.

Implementation & Technologies

What We Built

  • Real-time sensor data collection (soil moisture, rainfall, temperature)
  • Machine learning–based risk prediction model
  • Risk classification (Low, Medium, High)
  • Integrated system combining hardware and AI

Technologies Used

  • Python, NumPy, scikit-learn, joblib
  • Arduino and environmental sensors
  • Data processing and prediction pipeline
