// What We Built

THE SYSTEM

A complete end-to-end IoT pipeline — from raw sensor readings to real-time AI-powered risk classification, deployed live on the web.

01
📡
Sensor Collection
Real-time data from soil moisture, rainfall, humidity, and tilt sensors via Arduino hardware.
02
🧠
ML Risk Model
Scikit-learn classification model trained to predict LOW, MEDIUM, or HIGH landslide risk from sensor inputs.
03
⚡
FastAPI Backend
Python backend deployed on Render, receiving live sensor data and serving predictions via REST API.
04
🌐
Live Dashboard
Real-time web dashboard with alerts, sparklines, and risk visualization — accessible from any device worldwide.
